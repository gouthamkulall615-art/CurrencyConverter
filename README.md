#  Currency Converter

A lightweight web app to convert currencies in real time.

## Features
- Convert between major world currencies
- Real-time exchange rates via API
- Clean, minimal UI

## Tech Stack
- HTML / CSS / JavaScript
- Exchange Rate API (e.g., exchangerate-api.com or Open Exchange Rates)

## Getting Started

### Prerequisites
- Node.js (if using a framework)
- API key from your exchange rate provider
